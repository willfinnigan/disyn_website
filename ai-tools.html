Public RetroBioCat Platform

Our AI platform for retrosynthetic analysis of biocatalytic routes.

The RetroBioCat platform is a powerful tool for designing biocatalytic routes to target molecules. Originally an academic project, a publicly available version is available at retrobiocat.com, supported by Disyn Biotec.

RetroBioCat provides retrosynthetic analysis for biocatalytic routes, helping researchers identify potential enzymatic pathways to synthesize target molecules. The platform includes a comprehensive database of biocatalytic reactions and enzymes, making it a valuable resource for both academic and industrial researchers.

Key publications:

article Finnigan, W. et al. RetroBioCat as a computer-aided synthesis planning tool for biocatalytic reactions and cascades. Nat Catal 4, 98–104 (2021).

article Finnigan, W. et al. RetroBioCat Database: A Platform for Collaborative Curation and Automated Meta-Analysis of Biocatalysis Data. ACS Catal. 13, 11771–11780 (2023).

Private RetroBioCat Solutions

Secure, in-house implementations of RetroBioCat for enterprise use.

For clients who need a secure, in-house implementation of RetroBioCat, or programmatic access to its database and outputs, Disyn Biotec offers commercial licenses with the following benefits:

  • Software (as running on retrobiocat.com) as both source code and a ready to run docker container allowing quick set-up of an internal version of RetroBioCat
  • All services running locally, no internet connection required
  • A complete version of the RetroBioCat Database, available as a MongoDB file for immediate use with RetroBioCat, and as excel file for any other internal uses
  • Up to two days technical support for installation and training
  • EULA for permanent free-use internally, with complete ownership of all outputs (future updates not included)
  • Competitively priced
  • Customisation or additional features available on request
